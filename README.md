# Plant Disease Detection Dataset

Overview
This repository contains a curated image dataset of plant leaves used for detecting plant diseases using Machine Learning and Deep Learning models.

Classes Included
- Potato – Early Blight
- Potato – Late Blight
- Potato – Healthy
- Pepper Bell – Bacterial Spot

Dataset Structure
Images are uploaded in parts due to GitHub file limitations.
Each image belongs to one of the above classes.

Use Cases
- Image classification
- CNN-based plant disease detection
- Computer Vision projects
- Academic research

Note
Due to GitHub upload limits, the dataset is uploaded in batches.
For full training, images should be grouped class-wise.
